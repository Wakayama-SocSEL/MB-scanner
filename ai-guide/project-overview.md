# プロジェクト概要

## MB-Scanner
MB-Scanner は、GitHub 上の多数の公開 JavaScript リポジトリに対して、任意の CodeQL クエリを体系的かつ自動的に実行するためのバッチプラットフォームです。MB-search が生成したクエリや手動で作成したクエリの有効性を、実世界のコードベースで検証し、定量・定性的なデータを収集します。

## 主な機能

### 柔軟なターゲット検索
GitHub APIと連携し、言語、スター数、最終更新日といった豊富な条件を組み合わせて、解析対象となるリポジトリを柔軟に検索する。

### データ保存機能
検索結果を後続の処理で利用しやすいsqlite形式で保存する。

### CodeQL実行・連携機能
保存したリポジトリ情報を元に、CodeQL CLIを呼び出して解析を実行する。
