# 環境セットアップと依存管理

プロジェクトの環境構築や設定変更を行う際は、以下のガイドラインに従ってください。

## 参照ドキュメント
- [セットアップガイド](references/guide.md)

## 重要なルール
- **パッケージ管理**: 必ず **`uv`** コマンドを使用してください。`pip` や `poetry` は使用禁止です。
  - 追加: `uv add <package>`
  - 開発用追加: `uv add --dev <package>`
  - 同期: `uv sync`
- **タスクランナー**: コマンド実行には **`just`** を使用してください（例: `just fix`, `just typecheck`）。

## 一般的なワークフロー

### 1. 初回セットアップ / 環境リセット
環境がおかしい場合や初回構築時は以下を実行してください。
1. `references/guide.md` の「セットアップ手順」を確認する。
2. `.env` ファイルが存在するか確認し、なければ `.env.sample` からコピーして作成する。
3. `uv sync` を実行して依存関係をインストールする。

### 2. ライブラリの追加
1. `pyproject.toml` を確認する。
2. `uv add` コマンドでライブラリを追加する。
3. 必要に応じて `import` 文をコードに追加する。
