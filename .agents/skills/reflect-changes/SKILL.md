# ドキュメント反映・学習ワークフロー

実装タスクで得られた知見をプロジェクトの知識ベースに統合します。
**注意**: このスキルは、実装作業の履歴（チャットコンテキスト）ではなく、ファイルに保存された `plan.md` と `prompt.md` を情報源とします。

## 前提条件
- 直前に `/clear` を実行し、コンテキストが空の状態であること（推奨）。

## 手順

1. **コンテキスト収集**:
    - 以下のスクリプトを実行し、対象タスクの `plan.md` と `prompt.md` を読み込みます。
    ```bash
    # 引数なしの場合は最新のタスクを自動対象とする
    uv run python .claude/skills/reflect-changes/scripts/collect_context.py
    ```
2. **差分分析と更新**:
    - 読み込んだプランと実績に基づき、`.claude/ai-guide/` 内の関連ドキュメントを更新してください。
